LAPD Crime Analysis & Predictive Intelligence Project

Then sections:

Project Objective

To analyze crime occurrence data from the Los Angeles Police Department (LAPD) and identify patterns, trends, risk factors, and predictive indicators that can support crime prevention, resource allocation, and investigative decision-making.

Key Questions:
Which crime types occur most frequently?
Which locations experience the highest crime concentration?
What victim and incident characteristics are associated with crime patterns?
Can machine learning models identify factors linked to crime occurrence outcomes?
What trends and anomalies exist across crime categories?

Tools Used:
Python
Pandas
NumPy
Scikit-Learn
Matplotlib
Seaborn
Jupyter Notebook

Analytical Process:
Data Collection
Data Cleaning
Exploratory Analysis
Feature Engineering
Predictive Modeling
Evaluation
Insights & Recommendations

Results:
Random Forest achieved strong predictive performance.
Key predictors included location, area, victim demographics, and incident characteristics.
Several crime concentration patterns and hotspot trends were identified.
Findings demonstrated how data-driven approaches can support investigative and operational decision-making.
